# T14 Implementation Bundle

T14 implementa tiempos, productividad y comisiones en backend/API.

## Alcance

- `work_logs` como fuente de tiempos tecnicos.
- `work_log_events` como bitacora operativa de start/pause/resume/stop.
- `technician_commission_rules` como reglas de comision.
- Endpoints backend para start/pause/resume/stop/list y reglas.

## Fuera de alcance

- UI pesada.
- T15 reportes.
- WhatsApp provider.
- PWA push.
- `notification_events`.
- Inventario.
- Finanzas/caja/pagos.
